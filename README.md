# Crimes against causality


**An interactive mystery-solving book that trains readers to spot flawed scientific evidence.**

This repository contains materials for an ongoing book project by Simon Munzert and Dylan Thurgood.  
The book combines the logic of *detective puzzle books* with *scientific studies* to teach evidence literacy, critical thinking, and the scientific method.

**Read it online: <https://crimes-against-causality.github.io/>**

## Repository structure

The project spans two repositories:

- **this repo** (`crimes-against-causality.github.io`) — the website backbone: the
  [Quarto](https://quarto.org) book scaffold, the build script that turns the
  case content into chapters, styling, and the publishing workflow.
- **[`crimes-against-causality/cases`](https://github.com/crimes-against-causality/cases)** —
  the case content authors edit (one folder per case, plus the narrative).

The site is built from both by GitHub Actions and published to GitHub Pages.
A push to either repo redeploys it. See [`book/README.md`](book/README.md) for the
authoring and build workflow.
